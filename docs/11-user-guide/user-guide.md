# Руководство пользователя (кассира)

## Введение

**Авиакасса** — веб-приложение для оформления бронирований авиабилетов, управления платежами и формирования отчётов. Приложение предназначено для работы кассира авиакассы.

## Вход в систему

1. Откройте приложение в браузере по адресу `http://localhost`
2. На экране входа введите **логин** и **пароль**
3. Нажмите «Войти»

Учётные данные по умолчанию:

- Логин: `cashier1`
- Пароль: `password`

При успешном входе откроется страница поиска рейсов.

---

## Поиск рейсов

1. На странице поиска введите **Пункт отправления** (например, «Москва»)
2. Введите **Пункт назначения** (например, «Сочи»)
3. Нажмите «Найти рейсы»
4. В таблице отобразятся подходящие рейсы с информацией:
   - Номер рейса
   - Время вылета и прилёта
   - Доступные места
   - Стоимость билета

## Бронирование

1. В списке рейсов нажмите «Забронировать» напротив выбранного рейса
2. Заполните данные пассажиров:
   - **Имя** — имя пассажира
   - **Фамилия** — фамилия пассажира
   - **Номер паспорта** — серия и номер (5–20 символов, буквы и цифры)
   - **Дата рождения** — в формате ГГГГ-ММ-ДД
3. Для добавления ещё одного пассажира нажмите «+ Добавить пассажира» (максимум 9)
4. Для удаления пассажира нажмите «Удалить» рядом с его формой
5. Проверьте итоговую сумму
6. Нажмите «Подтвердить бронирование»

При успешном создании бронирования произойдёт переход на страницу оплаты.

**Ошибки при бронировании:**

- «Недостаточно свободных мест» — выберите другой рейс или уменьшите количество пассажиров
- «Поле обязательно для заполнения» — заполните все обязательные поля
- «Неверный формат паспорта» — проверьте номер (только буквы и цифры)

## Оплата

1. На странице оплаты выберите способ оплаты:
   - **Кредитная карта**
   - **Дебетовая карта**
   - **Банковский перевод**
2. Нажмите «Оплатить»
3. При успешной оплате откроется страница подтверждения с деталями билета

## Отмена бронирования

1. Перейдите на страницу бронирования через меню
2. Найдите нужное бронирование в списке
3. Нажмите «Отменить»
4. Подтвердите отмену в диалоговом окне

При отмене бронирования:
- Статус изменится на CANCELLED
- Места на рейсе будут возвращены в продажу

## Просмотр отчётов

1. Нажмите «Отчёты» в навигационном меню
2. Выберите период (начальная и конечная дата)
3. Нажмите «Сформировать»
4. Просмотрите сводку:
   - Общее количество бронирований
   - Сумма продаж
   - Распределение по статусам

## Выход из системы

1. Нажмите на имя пользователя в правом верхнем углу
2. Выберите «Выйти»
3. Сессия завершена, произойдёт перенаправление на страницу входа

---

## Устранение неполадок

- **Не удаётся войти** — проверьте логин и пароль. Убедитесь, что Caps Lock выключен
- **Рейсы не загружаются** — проверьте подключение к интернету и доступность сервера
- **Ошибка при бронировании** — проверьте заполнение всех полей и наличие свободных мест
- **Страница недоступна** — обратитесь к администратору
