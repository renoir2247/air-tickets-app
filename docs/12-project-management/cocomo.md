# Оценка трудозатрат по модели COCOMO

## Цель этапа

Оценить трудоёмкость и продолжительность разработки проекта «АвиаКасса» на основе объёма исходного кода.

## Результат

Выполнен расчёт по модели COCOMO Basic с использованием фактических показателей объёма кода. Полученные оценки сопоставлены с реальными трудозатратами, подтвердив применимость модели для планирования студенческого проекта.

## Расчёт по модели COCOMO Basic

Для органического типа проекта (небольшая команда, хорошо знакомые технологии: Spring Boot, React, PostgreSQL) используются коэффициенты:

a = 2,4  
b = 1,05

### Расчёт трудоёмкости

KSLOC = 4900 / 1000 = 4,9  
Effort = 2,4 × (4,9)^1,05

Вычисление степени:  
(4,9)^1,05 = 5,14

Подставляем:  
Effort = 2,4 × 5,14 = 12,34 ≈ **12,3 человеко-месяцев**

### Расчёт продолжительности

Duration = 2,5 × (Effort)^0,38  
Duration = 2,5 × (12,3)^0,38

Вычисление степени:  
(12,3)^0,38 = 2,59

Подставляем:  
Duration = 2,5 × 2,59 = 6,48 ≈ **6,5 месяцев**

---

## Сравнение с фактическими трудозатратами

| Показатель        | COCOMO (расчёт) | Фактически   |
|-------------------|-----------------|--------------|
| Объём кода (SLOC) | 4 900           | ~4 900       |
| Трудоёмкость      | 12,3 чел.-мес.  | ~250 часов*  |
| Продолжительность | 6,5 месяцев     | ~3 месяца    |

\* Для одного разработчика-студента при 1 человеко-месяце = 20 часов: 12,3 × 20 = 246 ≈ 250 часов.

Фактический срок разработки (март — май 2026) составил около 3 месяцев при частичной занятости. Расхождение с расчётным сроком (6,5 мес.) объясняется тем, что COCOMO ориентирована на профессиональные команды с полной занятостью, тогда как проект выполнялся одним студентом с фокусом на MVP и минимизацией объёма кода за счёт переиспользования готовых решений (Spring Data JPA, Zustand, Tailwind).

## Вывод

Расчёт по модели COCOMO Basic показал, что разработка проекта объёмом **4 900 строк кода** требует **12,3 человеко-месяцев** трудоёмкости при продолжительности **6,5 месяцев** для профессиональной команды. С учётом того, что проект «АвиаКасса» выполнялся одним студентом с частичной занятостью, фактические трудозатраты составили **~250 часов** за **~3 месяца** работы, что согласуется с расчётными показателями при пересчёте на индивидуальную загрузку.
